# GC Content Calculator

This is a simple Python program that calculates the GC content of a given DNA sequence.

## Features

1. Takes the DNA sequence from the user in the form of a string.
2. Converts the sequence to uppercase using the `.upper()` command.
3. Checks for invalid inputs, such as an empty string or characters other than `A`, `G`, `T`, and `C`.
4. Calculates the number of `G`s and `C`s and the length of the DNA sequence.
5. Calculates the GC content as a percentage.

## Example

**Input**
```text
ATGCGTACTA
```

**Output**
```text
The GC content of the DNA strand is 40.0%
```

## Formula Used

```text
GC Content = ((Number of G's + Number of C's) / Total Number of Bases) × 100
```
